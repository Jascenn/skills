# Skills

凌一的 Skills 集合 - 通用工具和自动化脚本

## Skills 列表

### wechat-mp-publisher

通用微信公众号创作与发布工具

**功能：**
- 话题创作（生成标题+大纲）
- 素材整理
- 推送草稿到公众号
- 保存到飞书 Base（可选）

**快速开始：**
```bash
cd wechat-mp-publisher/scripts
python3 init.py
```

[查看详细文档](./wechat-mp-publisher/README.md)

## 安装

```bash
git clone https://github.com/Jascenn/skills.git
cd skills/wechat-mp-publisher
```
