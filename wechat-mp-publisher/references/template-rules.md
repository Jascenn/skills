# Template Rules

## 正文母版规则

凌一公众号正文默认遵循以下规则：

- 正文不重复标题（标题只写在公众号草稿标题栏）
- 句号后换行，保持阅读节奏
- 块与块之间留一个空行
- 每块控制在 1–5 行，按语义切，不硬拆
- 风格口语化、第一人称、有判断，不写成教程说明书
- 推荐阅读单独成块
- 广告模块与固定结尾按默认模板插入
- **引号使用「」**：所有引号统一使用「」，不使用 ""
- **标点符号**：使用英文逗号（,），中文句号（。）
- **不重复标点**：避免过度使用标点符号（如：！！！、？？？）

## 空格使用规范

- **中英文之间加空格**：使用 JavaScript 编写
- **中文与数字间加空格**：花了 5000 元
- **数字与单位间加空格**：10 GB、16 GB
- **例外：度数、百分号不加空格**：90°、15%、35°
- **全角标点与其他字符间不加空格**

## 专有名词规范

使用正确的大小写：
- GitHub（不是 github 或 Github）
- JavaScript（不是 javascript）
- iOS（不是 ios）
- MacBook Pro
- HTML、CSS、Vue.js、React

## 推荐阅读格式

推荐阅读必须使用文本链接格式：

```md
推荐阅读：
[文章标题](https://mp.weixin.qq.com/s?...)
```

不要把推荐阅读写成一串裸 URL。

## 广告 / 固定结尾

默认包含以下内容：

- 副业避坑 CTA
- 广告正文
- 星球引导
- 围观群微信号
- 署名

除非用户明确要求修改，否则按默认模板处理。

## HTML 正文约束

- 正文单独保存为 HTML 文件
- 文件命名建议：`articles/YYYYMMDD_<关键词>_body.html`
- 段落样式以现有 `articles/mp_template.html` 为准
- 不要随意发明与母版不一致的新样式
